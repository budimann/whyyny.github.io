Halo, kali ini saya akan menjelaskan tentang bagaimana cara menggunakan Hexo. berikut langkah-langkahnya:

Membuat Akun dan Repository Github

langkah pertama buat akun CLICK HERE FOR GITHUB, jika kalian sudah Membuat akun andah bisa membuat Repository yang digunakan untuk membuat hexo WEB,
jika kalian tidak tau ini caranya dengan menggunakan gambar

Setelah itu lakukan hal yang seperti dibawah
yang gambar dibawah ada error saat penulisan name Repository itu tidak apa-apa karena itu nama yang sudah saya pakai, jadi jangan bingung

Install Node JS & Hexo

Download Node JS

Aslinya ada yang dari Tutorial saya yang membahas download hexo yaitu lewat
caranya Hexo Guide

jangan lupa ya cara install semua itu pake CMD
ini lah urutan cara Install hexo

jika kalian bingung “sudah keinstall apa belum” bisa juga dicek lewat

Deploy Hexo dan Apply Theme

Setelah itu buka CMD kalian/Anda kemudian arahkan kefolder tempat untuk menaruh folder WEB,
untuk Tutorial mengarahkan path CMD yaitu kalian bisa mencari sendiri di google.

Kemudian enter, untuk nama repository sudah pasti adalah tadi yang username-kalian.github.io.

kemudian buka browser kalian katik di tab url denga localhost:4000



Selanjutnya buka website Hexo Theme dan download salah satu tema yang kalian inginkan. Kalian bisa download tema lewat zip atau git. Perlu diketahui bahwa tiap tiap tema terdapat cara installnya sendiri dan mempunyai plugin sendiri. Jadi silahkan buka dan baca-baca wiki yang terdapat pada deskripsi tema. Jika kalian sudah mendownload tema, taruh folder tema ke dalam folder Themes pada folder web kalian. Silahkan masukkan plugin-plugin yang dibutuhkan untuk tema kalian, adapun jika tidak ada maka tidak usah. Untuk menginstall plugin sendiri caranya adalah memasukkan command plugin ke dalam Command Prompt di dalam folder website kalian tentunya.

Berikutnya adalah masuk ke bagian editing. Kalian bisa menggunakan software script editor untuk web seperti Atom, Dreamweaver, Notepad++ dll. Disini saya menggunakan Atom, kalian bisa download disini. Buka Atom atau script editor lain dan buka script config.yml yang terdapat di dalam folder website kalian. Pada bagian theme, isikan dengan nama tema kalian.



Silahkan save dan ketik ulang hexo server pada Command Prompt kemudian buka localhost:4000 maka tema kalian akan terpasang
