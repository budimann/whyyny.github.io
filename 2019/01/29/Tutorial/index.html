Tutorial

Jan 29, 2019

Halo, kali ini saya akan menjelaskan tentang bagaimana cara menggunakan Hexo. berikut langkah-langkahnya:

  1. Membuat Akun dan Repository Github
    Buat akun Github. Jika kalian sudah mempunyai akun Github sebelumnya, kalian bisa menggunakan akun tersebut asalkan akun tersebut belum digunakan untuk membuat Hexo web. Setelah mendaftar, buat sebuah repository dengan cara:
    New Repository -> isikan nama repository wajib dengan nama: (username-kalian).github.io -> Set Public -> Create repository.

Pada gambar diatas memang terdapat sebuah error, hal itu dikarenakan saya telah membuat repository dengan nama tersebut sebelumnya.

Langkah pertama telah selesai, selanjutnya adalah

  1. Install Node JS & Hexo
    Download dan install Node Js.

Kalian bisa membuka Hexo Guide untuk melihat command yang digunakan. Untuk install Hexo, masukkan command

1
npm install -g hexo-cli
Jika sudah, kalian bisa meng-check Hexo kalian dengan command

1
hexo -v
maka akan muncul versi Hexo kalian dll.

  1. Deploy Hexo dan Apply Theme
    Buka Command Prompt as Admin kemudian arahkan path ke folder tempat menaruh web yang akan kalian buat. Untuk tutorial mengarahkan path Command Prompt kalian bisa mencarinya di google. Jika sudah kalian arahkan, ketik command

1
hexo init nama-repository-kalian
Kemudian enter, untuk nama repository sudah pasti adalah tadi yang username-kalian.github.io.

Akan muncul progress connect Hexo kalian dengan repository. Jika sudah selesai, cobalah masukkan command

1
hexo server
Kemudian buka browser kalian dan ketikkan url localhost:4000 . Jika muncul tampilan seperti dibawah ini, artinya kalian telah berhasil.

Selanjutnya buka website Hexo Theme dan download salah satu tema yang kalian inginkan. Kalian bisa download tema lewat zip atau git. Perlu diketahui bahwa tiap tiap tema terdapat cara installnya sendiri dan mempunyai plugin sendiri. Jadi silahkan buka dan baca-baca wiki yang terdapat pada deskripsi tema. Jika kalian sudah mendownload tema, taruh folder tema ke dalam folder Themes pada folder web kalian. Silahkan masukkan plugin-plugin yang dibutuhkan untuk tema kalian, adapun jika tidak ada maka tidak usah. Untuk menginstall plugin sendiri caranya adalah memasukkan command plugin ke dalam Command Prompt di dalam folder website kalian tentunya.

Berikutnya adalah masuk ke bagian editing. Kalian bisa menggunakan software script editor untuk web seperti Atom, Dreamweaver, Notepad++ dll. Disini saya menggunakan Atom, kalian bisa download disini. Buka Atom atau script editor lain dan buka script _config.yml yang terdapat di dalam folder website kalian. Pada bagian theme, isikan dengan nama tema kalian.

Silahkan save dan ketik ulang hexo server pada Command Prompt kemudian buka localhost:4000 maka tema kalian akan terpasang.

Untuk men-deploy website kalian, masukkan command berikut berturut-turut

1
hexo generate –watch

1
hexo clean

1
hexo deploy
Jika sudah, silahkan buka url username-kalian.github.io di browser kalian. Jika muncul website, artinya langkah ketiga ini telah berhasil.

  1. Buat Post
    Untuk membuat post, masukkan command
    1
    hexo new post “judul-post-kalian”
    Post dan folder tempat menampung gambar akan terbuat di folder source/post/judul-post.

Di dalamnya kalian bisa mengetik sebuah content apapun terserah kalian.

Untuk menambahkan gambar, caranya adalah pindahkan gambar ke dalam folder yang parent dan namanya sama dengan post kalian. Jika sudah, copy kan nama gambar beserta ekstensinya. Kemudian pada script post, tambahkan
1

Alt maksudnya adalah text yang akan ditampilkan ketika user gagal memuat gambar.

  1. Buat Halaman About
    Untuk setiap tema, penerapan halaman about biasanya berbeda-beda. Jadi silahkan ikuti instruksi yang sudah disediakan pada wiki tema yang kalian pilih.
    Untuk tema yang saya pilih, pembuatan About sangatlah simple. Yaitu tinggal memasukkan command

1
hexo new page about
Silahkan isikan biografi kalian pada halaman About jika sudah berhasil dibuat.

Kira-kira begitulah tutorial ini, sekian sampai jumpa.
